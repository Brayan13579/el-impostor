# 🕵️ El Impostor

Juego multijugador de palabras en tiempo real. Todos reciben la misma palabra excepto el impostor, que sabe cuál es pero debe mezclarse con los demás.

## Cómo jugar
1. Todos se conectan al mismo link
2. Cada uno ve su palabra en privado
3. Dan pistas sin revelar la palabra
4. Votan quién es el impostor

## Deploy local
```bash
npm install
npm start
```

## Deploy en Render
- Build Command: `npm install`
- Start Command: `npm start`
